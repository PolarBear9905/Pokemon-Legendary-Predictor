# Probability of Legendary Pokemon - Website

This is a web application where users can enter the stats of the pokemon and it outputs the chance of having a <b>Legendary</b> Pokemon.

## About the Dataset

This Data was downloaded from Kaggle: <br> 
https://www.kaggle.com/datasets/abbas829/okmon-with-stats-complete-generation-1-6-pokedex
